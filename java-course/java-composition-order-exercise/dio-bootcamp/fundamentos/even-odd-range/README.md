# Exercício: Intervalo Par/Ímpar em Ordem Decrescente

## 📝 Descrição
Programa em Java que recebe dois números inteiros (onde o segundo deve ser maior que o primeiro) e uma opção (`par` ou `impar`).
O código exibe todos os números do tipo escolhido dentro do intervalo informado, **em ordem decrescente**, incluindo os limites.

## 💻 Tecnologias Utilizadas
- Java SE
- `Scanner` para entrada de dados via console
- Estruturas de controle: `do-while`, `for`, `if`
- Operadores lógicos (`&&`, `||`) e módulo (`%`)
